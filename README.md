# E2E Test Automation - 두부핑퐁 관리 대시보드

## 개요
두부핑퐁 관리 대시보드(`ppistaging.dubuhealth.in`)의 E2E 테스트 자동화 프로젝트

- **프레임워크**: Playwright + TypeScript
- **테스트 대상**: 스테이징 서버 (16개 관리 탭)
- **레포**: https://github.com/dubu-harry/e2e-automation

## 실행 방법

```bash
npm install
npx playwright install chromium
cp .env.example .env  # 계정 정보 입력

npx playwright test                              # 전체 실행
npx playwright test --grep @smoke                # 스모크 테스트만
npx playwright test --project=navigation-tests   # 네비게이션 테스트만
npx playwright test --project=page-tests         # 페이지별 테스트만
npx playwright test --project=session-tests      # 세션 테스트만
npx playwright show-report                       # HTML 리포트
```
